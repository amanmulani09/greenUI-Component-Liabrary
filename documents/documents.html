Buttons

Buttons are clickable elements by which user interacts with web interface and perform some action.

Default button:

To display specific category of button use class name such as btn-secondary, btn-success, btn-warning etc.

Outline button:

You believe in simplicity and hence want something which is not loud as background colors on button?

use btn-outline-* and implement the simplicity you believe in.

Disabled button:

We can make buttons look inactive by adding the disabled boolean attribute to any element.

Link button:

This button acts as a clickable link.

use btn-link ,btn-link-* to show clickable text functionality.

Primary Secondary

Floating Action button:

A floating action button (FAB) performs the primary, or most common, action on a screen. It appears in front of all screen content.

use btn-float for floating button.

Icon button:

Icon buttons mostly used in social media pages

use icon-btn class to replicate icon button as follows.

Images
